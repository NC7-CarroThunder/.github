## 프로젝트 기능

### 회원가입 / 소셜 로그인
- 카카오 소셜로그인을 통해 서비스를 이용할 수 있어요!

### 판매글 작성 / 수정
- 카테고리 설정 및 거래 유형을 선택해서 게시글을 올려보세요!

### 안전 결제
- 직접 결제가 어려우신가요? 캐럿썬더페이를 사용해보세요!

### 관심상품 등록
- 관심상품을 등록해보세요!

### 판매글 검색
- 지역, 카테고리 별로 검색이 가능해요!

### AI 번역 채팅 기능
- 번역할 언어를 설정하고 상대방과 채팅해보세요!
